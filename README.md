View your app in AI Studio: https://ai.studio/apps/62fab214-a493-4623-8f80-f879afe96833

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
